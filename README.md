# tinper-docs
开发者中心



## 使用方法

1. 全局安装`hexo-cli`

   ```
   $ sudo npm install -g hexo-cli
   ```

2. 下载仓库代码

   ```
   $ git clone git@github.com:iuap-design/tinper-docs.git && cd tinper-docs
   ```

3. 安装依赖

   ```
   $ npm install
   ```

4. 启动hexo服务，会占用4000端口

   ```
   $ hexo server
   ```

5. 访问[localhost:4000](localhost:4000)
